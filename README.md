# reGRC - GRC Tool

## Description 
A Governance Risk, and Compliance (GRC) platform designed to help organizations manage risks, maintain security controls, and gain operational visibility through a simple dashboard.

### Risk Repository
- Centralized repository for identifying and managing risks
- Track risk ownership, status, impact, and mitigation plans
- Maintain consistent risk documentation across the organization

### Control Library
- Maintain a centralized library of security and compliance controls
- Map controls to risks, frameworks, and regulatory requirements
- Standardize control documentation and ownership tracking

### Dashboard
- View organizational risk posture in real time
- Monitor control effectiveness and compliance status
- Visualize key risk and compliance metrics through dashboards
